HTML Semantic Reference

Semantic Elements

Tag Name Description
<header> Header The header element represents a container for introductory content or a set of navigational links.
<nav> Navigation The nav element defines a set of navigation links.
<section> Section The section element defines a section in a document.
<article> Article The article element specifies independent, self-contained content.
<aside> Aside The aside element defines some content aside from the content it is placed in (like a sidebar)
<footer> Footer The footer element defines a footer for a document or section.

Non Semantic Elements

Tag Name Description
<div> Div The div tag defines a division or a section in an HTML document.
<span> Span The span tag is an inline container used to mark up a part of a text, or a part of a document.

Note: Although these tags can be used similar to Semantic tags. They are considered Non Semantic because they tell us nothing about the content within.
